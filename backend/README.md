## Getting Started

### Requirements

- **Node** 24 ([nodejs.org](https://nodejs.org/))
- **pnpm** 10 ([pnpm.io](https://pnpm.io/))

### Installation

```bash
pnpm install
```

### Setup

1. Copy the `.env.example` file to a new `.env` file and fill out the environment variables.

### Running

```bash
pnpm dev
```

The server will run in [localhost:3000](http://localhost:3000).
